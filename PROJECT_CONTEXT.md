**Projeto:** Portal do Colaborador

# Objetivo

Desenvolver um sistema web para gerenciamento de solicitações de folgas dos colaboradores.

---

Frontend

* React
* Next.js
* TypeScript
* Tailwind CSS
* shadcn/ui

Backend

* Supabase
* PostgreSQL
* Auth
* Storage
* Row Level Security

Infraestrutura

* GitHub
* Vercel

# Documentação Oficial

A pasta `docs/` contém toda a documentação oficial do projeto.

Nenhuma regra deverá ser alterada sem atualização da documentação.

# Arquitetura

* Frontend em React/Next.js.
* Backend no Supabase.
* Banco PostgreSQL.
* Autenticação pelo Supabase Auth.
* Controle de acesso por RLS.

---
