# auth

Responsável pela autenticação e autorização de usuários no sistema.

**Subdomínio:** Identidade e Acesso

**Responsabilidades:**
- Login/logout com Supabase Auth
- Gerenciamento de sessão
- Redefinição de senha (admin)
- Primeiro acesso (cadastro de senha)
- Middleware de proteção de rotas
- Identificação de roles do colaborador via RBAC (colaborador_roles)
