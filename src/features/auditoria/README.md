# auditoria

Registro e consulta de trilha de auditoria.

**Subdomínio:** Governança

**Responsabilidades:**
- Registro de todas as operações relevantes
  (criação, alteração, exclusão, aprovação, reprovação)
- Consulta com filtros (período, colaborador, ação, entidade)
- Exportação de auditoria
- Imutabilidade dos registros
