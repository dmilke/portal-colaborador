# administracao

Painel administrativo do sistema.

**Subdomínio:** Governança

**Responsabilidades:**
- Configurações globais do sistema
- Gestão de regras de negócio (fechamento mensal, antecedência máxima, limite de solicitações)
- Gerenciamento de tipos de folga
- Gerenciamento de datas bloqueadas
- Relatórios gerenciais
- Dashboard administrativo
