# documentos

Gestão de documentos institucionais e do colaborador.

**Subdomínio:** Conteúdo

**Responsabilidades:**
- Upload e armazenamento de documentos (Supabase Storage)
- Tipos documentais (regulamentos, formulários, contratos)
- Controle de versão
- Permissões baseadas em RBAC (roles)
- Associação a colaboradores, departamentos ou unidades
