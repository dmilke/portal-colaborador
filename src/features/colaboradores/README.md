# colaboradores

Gestão do cadastro de colaboradores.

**Subdomínio:** Pessoas

**Responsabilidades:**
- Cadastro, edição e consulta de colaboradores
- Vinculação a departamento, cargo e unidade
- Histórico de lotação (mudanças de departamento)
- Controle de acesso via RBAC (roles: admin, colaborador)
- Relacionamento com `auth.users` via `auth_user_id`
