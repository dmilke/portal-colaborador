# 11 - Prompts para IA

**Projeto:** Portal do Colaborador

**Versão:** 1.0.0

**Status:** CONGELADO

---

# Objetivo

Padronizar a utilização de Inteligência Artificial durante o desenvolvimento do Portal do Colaborador.

---

# Prompt Geral

Antes de implementar qualquer funcionalidade:

* leia o PROJECT_CONTEXT;
* siga todos os documentos oficiais;
* preserve a arquitetura;
* nunca altere regras de negócio;
* nunca altere nomenclaturas oficiais;
* nunca crie tabelas fora do Modelo de Dados;
* utilize React, Next.js, TypeScript, Tailwind CSS, shadcn/ui e Supabase.

---

# Prompt para Correção

Corrija exclusivamente o problema informado.

Não altere outras funcionalidades.

Preserve a compatibilidade com o restante do sistema.

---

# Prompt para Refatoração

Refatore apenas o código solicitado.

Não altere comportamento funcional.

Não modifique regras de negócio.

---

# Prompt para Banco de Dados

Gerar exclusivamente migrações SQL.

Não alterar tabelas manualmente.

Toda alteração deverá ser versionada.

---

# Prompt para Testes

Gerar testes considerando:

* requisitos funcionais;
* regras de negócio;
* requisitos não funcionais.

---

# Prompt para Código

Gerar código:

* limpo;
* reutilizável;
* tipado;
* documentado;
* compatível com ESLint;
* compatível com Prettier.

---

# Prompt para Revisão

Antes de responder:

* verificar conformidade com a documentação;
* verificar impacto nas regras de negócio;
* verificar impacto na arquitetura.

---
