# 09 - Wireframes

**Projeto:** Portal do Colaborador

**Versão:** 1.0.0

**Status:** CONGELADO

---

# Objetivo

Documentar a estrutura das telas do Portal do Colaborador.

Os wireframes representam a organização funcional das telas, não o layout visual definitivo.

---

# Tela 01 - Login

## Componentes

* Logo da Instituição
* Campo Usuário
* Campo Senha
* Botão Entrar

---

# Tela 02 - Dashboard do Colaborador

## Cabeçalho

* Nome do colaborador
* Sino de notificações
* Menu do usuário

## Menu

* Dashboard
* Solicitar Folga
* Histórico
* Perfil

## Conteúdo

* Calendário
* Próximas solicitações
* Últimas notificações
